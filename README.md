# DETONA — Puzzle Roguelike 🎮

DETONA é um jogo web no estilo puzzle roguelike, desenvolvido com HTML, CSS e JavaScript.

O projeto possui uma interface moderna, sistema de níveis, pontuação, peças, bombas, chefes e melhorias durante a partida.

## Tecnologias utilizadas

- HTML
- CSS
- JavaScript

## Funcionalidades

- Jogo em navegador
- Sistema de pontuação
- Progressão de níveis
- Boss a cada 5 níveis
- Peças especiais com bomba
- Escolha de melhorias durante a partida
- Interface responsiva
